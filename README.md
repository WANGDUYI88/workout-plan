# Workout Plan

一个静态的居家肩臂训练计划页面，包含训练计划和器材选购指南。

线上地址：https://wangduyi88.github.io/workout-plan/

## 使用方式

- 直接用浏览器打开 `index.html`
- 或者在有 Node.js 的环境中运行 `npm start`，再打开终端显示的本地地址

## 页面

- `index.html`：训练计划主页
- `equipment.html`：器材选购指南

## 目录

- `assets/styles.css`：共享页面样式
- `assets/app.js`：训练阶段切换与滚动动画逻辑
- `assets/*.jpg`：动作示意图
- `assets/motion/*.webp`：正式页面使用的动作动图
- `assets/wall-pushup-guide-v2.png`：早期墙壁俯卧撑图示草案，正式页面未引用
